# INTERACTIVE-QUIZ-APPLICATION

## PROJECT OVERVIEW

The Interactive Quiz Application is a web-based quiz system developed using HTML, CSS, and JavaScript. The application presents multiple-choice questions to users, provides instant feedback on answers, and calculates the final score at the end of the quiz.

The project demonstrates core front-end development concepts including dynamic content loading, event handling, DOM manipulation, and user interaction.


## FEATURES

- Dynamic Question Loading
- Multiple Choice Questions
- Instant Feedback for Answers
- Score Calculation
- Final Result Display
- Restart Quiz Functionality
- User-Friendly Interface
- Responsive Design

## TECHNOLOGIES USED

- HTML5
- CSS3
- JavaScript 

## HOW TO RUN THE PROJECT

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
4. Start answering the quiz questions.
5. View your score after completing the quiz.

## WORKFLOW

1. Questions are loaded dynamically from a JavaScript array.
2. Users select an answer from the available options.
3. Instant feedback is provided for each selection.
4. Scores are updated automatically.
5. Final results are displayed after all questions are answered.
6. Users can restart the quiz and attempt it again.

## OUTPUT

The application displays:

- Quiz Questions
- Multiple Choice Options
- Correct/Wrong Feedback
- Final Score
- Restart Button


## CONCLUSION

This project successfully implements an interactive quiz application using front-end technologies. It provides an engaging user experience through dynamic question loading, instant feedback, and score tracking while demonstrating practical JavaScript programming concepts.
